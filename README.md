
# Profiles App (Lab 1)

A simple React app built with Vite and React-Bootstrap.  
Displays a list of profiles with Like buttons and allows adding new profiles via a controlled form with validation.

## Getting Started

 ```bash
 npm install
 npm run dev
```

## Build

```bash
npm run build
```
